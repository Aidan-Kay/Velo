## Copilot Agents (`.github/agents/`)

### Standalone Agents

| File                                    | Role                                                                                                                                   |
| --------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| `.github/agents/app-designer.agent.md`  | Expert application designer — real-time UI, component systems, product thinking                                                        |
| `.github/agents/code-reviewer-agent.md` | Standalone code reviewer — 4-pillar review (correctness, performance, resilience, architecture), writes reports to `.md/code-reviews/` |
| `.github/agents/todo-implementer.md`    | Reads `.md/todo.md` and implements tasks, writes reports to `.md/implementation-reports/`                                              |
| `.github/agents/ui-reviewer.agent.md`   | UI/UX design auditor — Nielsen heuristics, visual hierarchy, accessibility, information architecture                                   |

### Workflow Agents

Orchestrated pipeline triggered by sending requirements to the **orchestrator**. The orchestrator handles both planning (interactive clarification + plan drafting in plan mode) and coordination of implementation subagents. All artefacts written to `.md/workflow-artifacts/YYYY-MM-DD_<slug>/`.

| File                                   | Role                                                                                                                                                                                        |
| -------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `.github/agents/orchestrator.agent.md` | Combined planner + orchestrator (plan mode) — gathers context, clarifies requirements with user, produces plan, then automatically drives implementer → reviewer → security → docs pipeline |
| `.github/agents/implementer.agent.md`  | Implements code changes per approved plan, handles reviewer/security fix feedback                                                                                                           |
| `.github/agents/reviewer.agent.md`     | Read-only code reviewer — evaluates implementation against plan's acceptance criteria, returns PASS/FAIL verdict                                                                            |
| `.github/agents/security.agent.md`     | Read-only security auditor — OWASP Top 10 + Electron-specific checks, returns PASS/FAIL verdict                                                                                             |
| `.github/agents/docs.agent.md`         | Updates knowledge base files per CLAUDE.md documentation rules                                                                                                                              |
